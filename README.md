# Graphic Design Portfolio

## Overview
This repository showcases my graphic design work created using **Canva**. The project focuses on building a visual identity and marketing materials for a local **Eyewear Store**, demonstrating my ability to combine creativity with commercial goals.

## Project: Eyewear Store Branding
**Role:** Graphic Designer
**Tools Used:** Canva

### Scope of Work
- Designed engaging social media posts to showcase new eyewear collections.
- Created promotional banners for seasonal offers and discounts.
- Maintained a consistent visual theme that reflects elegance and clarity.
